📖 README / 開発ガイド

このページでは、 むらびと君(アドオン用) のアドオン開発方法について説明します。 テンプレートを利用することで、 簡単に新しいアドオンを作成できます。

🌱 アドオンとは

addon/ フォルダ内へPythonファイルを配置すると、 Bot起動時に自動検出され、 load_addons() によって読み込まれます。 アドオンを利用することで、 Bot本体を書き換えることなく 自由に機能を追加できます。

💡 ポイント

アドオンは Discord.pyのCog として動作します。 既存機能を壊さず、 独立した機能として追加できます。

📝 アドオン作成手順

Step 1 テンプレートをコピーする

まずは template/ フォルダ内にあるテンプレートを コピーします。 

template/
└── sample_addon.py

コピーしたファイル名を、 作成したいアドオン名へ変更してください。 

my_addon.py
💡 おすすめ

初めてアドオンを作成する場合は、 テンプレートをそのままコピーして 編集する方法がおすすめです。

Step 2 コードを書く

コピーしたテンプレートを編集し、 必要な機能を追加します。

例えば次のような機能を追加できます。

  • スラッシュコマンド
  • 通常コマンド
  • イベントリスナー
  • データベース処理
  • 独自ロジック

以下はスラッシュコマンドの例です。 

@app_commands.command(
    name="sample",
    description="サンプルコマンド"
)

async def sample(
    self,
    interaction: discord.Interaction
):

    await interaction.response.send_message(
        "サンプルです",
        ephemeral=True
    )
⚠️ 注意

setup(bot) は削除しないでください。 この関数が存在しない場合、 アドオンは読み込まれません。

Step 3 addonフォルダへ配置する

コードの作成が完了したら、 作成したPythonファイルを addon/ フォルダへ配置します。 

VillageCore-main/
│
├── addon/
│   ├── sample_addon.py
│   └── my_addon.py ← 作成したアドオン
│
└── bot.py
💡 ポイント

Botは起動時に addon/ フォルダ内のアドオンを自動検出し、 順番に読み込みます。

Step 4 Botを再起動する

アドオンを配置したら、 Botを再起動してください。 

python bot.py

正常に読み込まれると、 起動ログへ次のように表示されます。 

✅ loaded addon: addon.my_addon
💡 読み込みについて

Addonモジュールは bot.py 起動時に自動検出され、 load_addons() によって読み込まれます。

Step 5 動作確認

Botの再起動後は、 以下の内容を確認してください。

  • Botが正常に起動している
  • 起動ログにエラーが出ていない
  • アドオンが読み込まれている
  • Discordでコマンドが表示される
  • 期待した動作になっている

✨ アドオンで追加できる機能

アドオンでは、 Bot本体を編集することなく さまざまな機能を追加できます。

  • 🧩 スラッシュコマンド
  • 💬 通常コマンド
  • 🎧 イベントリスナー
  • 🗄️ SQLiteなどのデータベース処理
  • 🪙 コイン・ポイントシステム
  • ⚙️ サーバー専用機能
  • 📨 自動メッセージ送信
  • 🎮 独自ゲーム機能

🚫 アドオンの読み込みを無効化する

読み込みたくないアドオンは、 環境変数 DISABLED_ADDONS で指定できます。

1つだけ無効化する 

DISABLED_ADDONS=addon.sample_addon

複数無効化する 

DISABLED_ADDONS=addon.sample_addon,addon.my_addon
⚠️ 注意

アドオン名は addon.ファイル名 の形式で指定してください。 拡張子 .py は不要です。

⚠️ よくあるミス

アドオンが正常に読み込まれない場合は、 以下を確認してください。

  • setup(bot) を書き忘れている
  • ファイル名と読み込み名が一致していない
  • addon/ フォルダ以外に配置している
  • Pythonのimportに間違いがある
  • Botを再起動していない
  • 必要なライブラリがインストールされていない

📄 templateフォルダについて

template/ フォルダには、 アドオン作成用のテンプレートファイルが 保存されています。

新しいアドオンを作成するときは、 テンプレートをコピーして編集してください。 

template/

└── sample_addon.py
💡 テンプレートを利用するメリット

テンプレートには、 アドオンに必要な基本構成が あらかじめ用意されています。

以下の内容を最初から利用できます。
  • Discord.py の import
  • Cog の基本構造
  • setup(bot)
  • サンプル処理

🧩 アドオンテンプレート例

基本的なアドオンは 以下のような形になります。 

import discord

from discord import app_commands

from discord.ext import commands



class MyAddon(commands.Cog):

    def __init__(self, bot: commands.Bot):

        self.bot = bot



    @app_commands.command(
        name="sample",
        description="サンプルコマンド"
    )

    async def sample(
        self,
        interaction: discord.Interaction
    ):

        await interaction.response.send_message(
            "サンプルです",
            ephemeral=True
        )



async def setup(bot: commands.Bot):

    await bot.add_cog(
        MyAddon(bot)
    )

  • ☑ Discord上で動作確認した
