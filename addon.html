アドオン作成方法

むらびと君(アドオン用)では、Pythonファイルを追加することでアドオンを作成できます。

作成手順

  1. /addon フォルダへ *.py ファイルを追加します。
  2. 各ファイルに async def setup(bot) または def setup(bot) を定義します。
  3. bot.load_extension("addon.<module_name>") 形式で読み込まれます。

サンプル

from discord.ext import commands

class MyAddon(commands.Cog):

    def __init__(self, bot: commands.Bot):
        self.bot = bot

    @commands.Cog.listener()
    async def on_ready(self):
        print("MyAddon loaded")


async def setup(bot: commands.Bot):
    await bot.add_cog(MyAddon(bot))

設定

読み込み除外は DISABLED_ADDONS 環境変数で設定できます。 

DISABLED_ADDONS=addon.sample_addon

Addon読み込みについて

Addon モジュールは bot.py の起動時に自動検出され、 load_addons() で読み込まれます。
